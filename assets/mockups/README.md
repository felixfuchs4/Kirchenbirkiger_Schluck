# Mockups

UI-Mockups und Wireframes für die Bedien- und Anzeigeoberfläche.

## Namenskonvention

```
Mockup_<Bereich>_<Screen>_v<Version>.<ext>
```

Beispiele:
- `Mockup_Bedien_Spielplan_v1.png`
- `Mockup_Anzeige_Spielstand_v1.png`
- `Mockup_Anzeige_Infoscreen_v2.pdf`

## Geplante Screens

### Bedienoberfläche

| Screen | Beschreibung |
|---|---|
| Spielplan | Übersicht aller Spiele mit Status und Reihenfolge |
| Ergebniseingabe | Duell-für-Duell-Erfassung mit Versuchslogik |
| Gruppenrangliste | Live-Tabelle mit Punkten, Siegen, Direktvergleich |
| Korrekturen | Nachträgliche Änderungen an Versuchen und Spielern |

### Anzeigeoberfläche (Beamer)

| Screen | Beschreibung |
|---|---|
| Spielstand | Live-Anzeige laufende Partie inkl. Duellstand |
| Infoscreen | Letzte Begegnung, nächste Begegnung, aktuelle Tabelle |
| Gruppenrangliste | Tabellenstand aller Gruppen |
| Bracket | K.O.-Baum der Finalrunde |
| Siegeranzeige | Abschlussanzeige mit Sieger und Endstand |

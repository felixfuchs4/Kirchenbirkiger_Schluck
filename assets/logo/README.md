# Logo

Projektlogo für den Kirchenbirkiger Schluck.

## Dateien

| Datei | Format | Verwendung |
|---|---|---|
| `Logo_Kirchenbirkiger_Schluck.png` | PNG | Anwendungsfenster, Anzeigescreen, Druckmaterialien |

## Verwendung in der Anwendung

Das Logo wird im Anzeigescreen (Beamer-Ansicht) eingeblendet – insbesondere auf dem Infoscreen
zwischen den Spielen. In der Bedienoberfläche erscheint es in der Titelzeile.

## Hinweise

- Möglichst als PNG mit transparentem Hintergrund bereitstellen.
- Mindestauflösung für Beamer-Ansicht: 300 × 300 px.

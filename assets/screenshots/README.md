# Screenshots

Screenshots der laufenden Anwendung zur Dokumentation und Kommunikation.

## Namenskonvention

```
Screenshot_<Bereich>_<Screen>_v<Version>_<Datum>.png
```

Beispiele:
- `Screenshot_Bedien_Spielplan_v1.0_2026-06-12.png`
- `Screenshot_Anzeige_Spielstand_v1.0_2026-06-12.png`

## Wann Screenshots ablegen?

- Nach Fertigstellung eines neuen Screens
- Bei größeren UI-Änderungen (vorher/nachher)
- Vor einem Release als Zustandsdokumentation

## Hinweis

Screenshots werden ergänzt, sobald die UI-Implementierung (Schritt F laut
[Implementierungsstand](../../docs/Implementierungsstand.md)) abgeschlossen ist.

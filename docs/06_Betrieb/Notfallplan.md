# Notfallplan

## Mögliche Probleme und Lösungen

### Anwendung startet nicht

1. .NET Runtime prüfen
2. Fehlermeldung notieren
3. Neustart versuchen
4. Fallback: Turnier manuell auf Papier führen

### Anwendung stürzt ab

1. Zuletzt gespeicherte Datei laden
2. Falls kein Speicherstand: Ergebnisse aus Gedächtnis rekonstruieren
3. Anwendung neu starten

### Anzeigescreen funktioniert nicht

1. HDMI-Verbindung prüfen
2. Bildschirm-Erkennung im Betriebssystem prüfen
3. Nur mit Bedienoberfläche weitermachen, Ergebnisse verbal ansagen

### Datei ist beschädigt

1. Backup aus `quellen/` prüfen
2. Ergebnisse manuell neu eintragen

## Notfallszenarien

### Anwendung stürzt ab

1. Anwendung neu starten
2. Letztes Backup laden (automatisch oder manuell auswählen)
3. Ggf. letztes abgeschlossenes Spiel neu eintragen

### Datei beschädigt oder nicht ladbar

1. Älteres Backup aus dem Backup-Ordner laden
2. Falls kein Backup vorhanden: Ergebnisse manuell rekonstruieren

### Zweiter Bildschirm fällt aus

1. HDMI-Verbindung prüfen
2. Bildschirmerkennung in Windows prüfen (erweiterter Betrieb aktiv?)
3. Turnier mit nur einem Bildschirm (Bedienoberfläche) fortführen
4. Ergebnisse verbal oder per Aushang kommunizieren

### Laptop fällt aus

1. Backup-Kopie auf USB-Stick vorhanden?
2. Ersatzgerät mit installierter Anwendung verfügbar?
3. Fallback: Turnier manuell auf Papier fortführen

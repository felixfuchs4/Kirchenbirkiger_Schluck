# Checkliste Turniertag

## Vorbereitung (1 Tag vorher)

- [ ] Anwendung auf aktuellem Stand
- [ ] Turnierdaten angelegt und geprüft
- [ ] Laptop geladen
- [ ] Backup der Turnierdaten erstellt

## Aufbau (vor Ort, vor Turnierbeginn)

- [ ] Hardware aufgebaut ([Aufbau_vor_Ort.md](Aufbau_vor_Ort.md))
- [ ] Anwendung gestartet
- [ ] Anzeigescreen funktioniert
- [ ] Testlauf mit Beispieldaten durchgeführt
- [ ] Teams und Gruppen eingetragen

## Während des Turniers

- [ ] Regelmäßige Zwischenspeicherung
- [ ] Ergebnisse zeitnah eintragen
- [ ] Anzeigescreen aktuell halten

## Nach dem Turnier

- [ ] Endergebnis dokumentiert
- [ ] Datei gesichert (USB / Cloud)
- [ ] Hardware abgebaut
- [ ] Feedback notiert

## Notfallkontakte

> Siehe [Notfallplan.md](Notfallplan.md).

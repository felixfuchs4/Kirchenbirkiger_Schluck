# Archiv

Veraltete oder abgelöste Dokumente, die zur Nachverfolgung aufbewahrt werden.
Aktive Dokumente befinden sich **nicht** hier.

## Wann wird hier archiviert?

- Dokumente, die durch eine neuere Version ersetzt wurden
- Konzeptansätze, die verworfen wurden (mit Begründung)
- Entscheidungen, die revidiert wurden

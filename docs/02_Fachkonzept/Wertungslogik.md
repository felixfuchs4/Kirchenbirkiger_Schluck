# Wertungslogik

## Punkte pro Ergebnis (Tabellenwertung)

### Standardmodus: Eishockey-System

| Ergebnis | Punkte Sieger | Punkte Verlierer |
|----------|--------------|-----------------|
| Sieg in regulärer Spielzeit | 3 | 0 |
| Sieg nach Stechen / Overtime | 2 | 1 |

### Alternativer Modus: Einfaches System (bei Turniererstellung wählbar)

| Ergebnis | Punkte Sieger | Punkte Verlierer |
|----------|--------------|-----------------|
| Sieg | 1 | 0 |

Es gibt kein Unentschieden auf Partielebene – immer hat eine Mannschaft gewonnen.

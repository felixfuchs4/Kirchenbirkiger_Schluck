# Turniermodus und Wertung

Dieses Dokument beschreibt die unterstützten Turniermodi und die Wertungslogik.

## Turniermodi

### V1 – Zwei-Gruppen-Modus

- Zwei Gruppen, Gruppenphase (jeder gegen jeden innerhalb der Gruppe)
- Anschließende Platzierungsspiele

### Geplante Modi (V2+)

Je nach Teilnehmerzahl soll die Anwendung passende Turniermodi vorschlagen:

| Teilnehmer | Beispielmodus |
|-----------|--------------|
| 4–6 | Gruppenphase + Finale |
| 7–12 | Gruppenphase + Halbfinale + Finale |
| 13+ | Gruppenphase + Viertelfinale + Halbfinale + Finale |

> Die genaue Zuordnung wird später im Fachkonzept festgelegt.

## Wertung

### Punkte pro Spiel

Das Wertungssystem wird bei der Turniererstellung festgelegt. Zwei Modi stehen zur Auswahl:

**Standard: Eishockey-System**

| Ergebnis | Punkte |
|----------|--------|
| Sieg in regulärer Spielzeit | 3 |
| Sieg nach Stechen / Overtime | 2 |
| Niederlage nach Stechen / Overtime | 1 |
| Niederlage in regulärer Spielzeit | 0 |

**Optional: Einfaches System**

| Ergebnis | Punkte |
|----------|--------|
| Sieg | 1 |
| Niederlage | 0 |

Unentschieden ist auf Partielebene nicht vorgesehen.
